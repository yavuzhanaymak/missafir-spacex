
# SpaceX Launchs App




## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/yavuzhanaymak/missafir-spacex
```

Proje dizinine gidin

```bash
  cd missafir-spacex
```

Gerekli paketleri yükleyin

```bash
 yarn install
```

Sunucuyu çalıştırın

```bash
  npm run start && npm run ios
```
